# RS-DAT -- Remote Sensing - Deployable Analysis Environment
Welcome to the RS-DAT GitHub organization!

RS-DAT (Remote Sensing Deployable Ananlysis Enviornment) is a project by the Netherlands eScience Center in colaboration with SURF.
RS-DAT provides researchers with the means to seamlessly scale their analyses and workflows in the era of big data while maintaining their accustomed
working environment.
